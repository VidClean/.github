# VidClean

Free video, audio, and image tools for content creators.
No account, no watermark, no monthly cap.

## 26 free tools including

- AI transcription with no monthly minutes cap (vs Otter's 300 min/month limit)
- Background noise removal powered by DeepFilterNet3
- Silence removal
- AI thumbnail creator with face upload
- Add subtitles with 10 font styles
- AI chapters and description generator
- And more at [vidclean.net](https://vidclean.net)

## Why free

We self-host Parakeet-TDT, Whisper, and DeepFilterNet3 on CPU workers for
about $50/month. No per-minute API bills means no reason to cap you.

Built by one person alongside a day job.
